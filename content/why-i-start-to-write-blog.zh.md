# 我为什么开始写博客了

美国神经科学家大卫·伊葛门（David Eagleman）曾在他的著作《死后四十种生活》（Sum: Forty Tales from the Afterlives）中大致提到过这样的观点：

> 人一共有三次死亡，
> 1. 第一次死亡： 当你的心跳停止，呼吸消逝，你在生物学上被宣告了死亡。 
> 2. 第二次死亡： 当你下葬，人们穿著黑衣出席你的葬礼，你在社会上不复存在，你从人际关系网里消逝。
> 3. 第三次死亡： 是这个世界上最后一个记得你的人，把你忘记，于是，你就真正地死去，整个宇宙都将不再和你有关。



我倾向于这样描述：除了 DNA 序列和社会关系，是不同的经历与记忆把你和其他人区别开来。
总之，按照这样的定义，像艾伦图灵这样的人就只死过两次。
我也想实现第三次永生。但是我没破解纳粹密码的能力，大概只有阉割自己和下毒自杀的能力。[^Turing]

所以，我妄图通过留下一些文字记录来实现这个目标。

## 现实意义

宏大叙事就到此为止吧，下面是一些更实际的意义。

**释放表达欲**：一天不锐评我浑身上下细胞都在痒。

**练笔**：~~不管是母语还是英语，写着写着总能变好点吧 。~~ 至少可以积累撰写提示词的经验。

**不再做“差不多先生”**：整个博客是一款需要长期投入精力的产品，而每篇博文都可以算作其中一项可以反复打磨雕琢的产品特性。通过打造这个产品我想逼自己长期把一件事做好、维护好。

**认识一些有趣的人**：说不定就被谁认知到了呢。

## AI 时代怎么写博客

**少写“客观”多聊“主观”：** 坚决避免写那些 AI 一秒千字的内容。反而应该多输出自己的想法、感受和活生生的经历。

**主动拥抱工具：** 积极利用 AI 去查资料，做翻译和文本润色，提升文字质量。他们 Gemini 真的太厉害了，现在手机里面什么事儿都能问，有什么问题直接在 Gemini 里面问，答案很快就到你屏幕窗口。[^Quick]


## 为什么不在平台连载
不在内容平台上写，主要是基于以下考虑

**去商业化与自由：** 互联网平台往往带有强烈的商业化和变现倾向。如果写作的初衷并非为了引流变现，那么在平台上持续输出，总感觉就像是为互联网企业打白工。


**高可控性与可迁移性：** 平台改规则甚至关停都难以预料，到时候在上面写的东西想搬走都难。而 GitHub 的商誉那比他们不知道高到哪里去了，所以我用 Markdown 写完往 GitHub Pages 上传。即使有天封杀所有中国用户，我在本地也还有备份。

Linus 说他的东西都直接丢网上，如果有价值的话很多人会帮他保存好。[^Linus] 我只能靠自己。

## 计划
目前确定的更新频率是平均每个季度至少要产出一篇文章。

- 每年肯定会有一篇“年度追番点评”。毕竟，以现在动画业界的状态，一年也就够憋出一篇了。
- 另外会记载一些我觉得值得记录的大事件，不论技术领域的还是公共领域的。
- 以及任何发布出来不会给我造成麻烦的内容。
