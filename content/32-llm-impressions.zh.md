# AI 及产品使用体验

不到两年过去，除中美两国外其他的大模型已经无力在一线竞争。基本只在合规，细分领域和本地语言有一些优势了。

目前的主力
+ 网页版: ChatGPT
+ Code Agent: Claude Code with DeepSeek V4 Pro

## 海外御三家

### [ChatGPT](https://chatgpt.com/)
- 网页版基本不排队这点好评
- 非常吹毛求疵和爱抬杠的一个模型，他的回答就跟审查论文一样，一丁点小细节都要纠正
- 据说奥特曼故意让别的模型蒸馏，染上了「我就在这里不逃不躲稳稳的接住你」之后，自己就修复了

### [Gemini - chat to supercharge your ideas](https://gemini.google.com/)
- 网页版基本不排队这点好评，不过历史纪录常常会消失
- 理论上联动Google 的邮件，云盘，日历效果会更好，不过我用不上
- 美国糖包，对用户只有全肯定。相比被拍马屁我宁可被抬杠，暂时弃用。

### [Claude](https://claude.ai/new)
- 用不起模型，只用 Claude Code，工程化这方面确实还不错
- 通用领域基本已经摆烂了，目前专精编程领域
- 在封杀中国用户这方面跟 tiktok 能扳手腕
- 隔三差五无底线炒作

### [Grok](https://grok.com/)
- 这个在高峰期会触发排队
- 听说成人内容限制比较少

## 国内产品

### [DeepSeek](https://chat.deepseek.com/)
- V4-Pro只能说是模型守门员，不过这么便宜的 token 还要什么自行车了。

### [Mimo](https://mimo.xiaomi.com/zh/)
- DeepSeek 多模态版本

其他的豆包和元宝基本公认痴呆

下面写于 2024-10-30

## [ChatGPT](https://chatgpt.com/)

- 大模型的比特币
- 额度感觉也用的挺快的。不过额度跑完以后会切换到免费模型，应该是 3.5

## [Claude](https://claude.ai/new)

- 或许是大模型的以太坊？和上面那位互为犄角之势
- 免费额度用的感觉还是偏快的，大约 4 小时刷新
- 有一个新的 CodePad，有意思

## [tongyi.aliyun.com](https://tongyi.aliyun.com/qianwen/)

- 渐变色感觉不好看，但是总体比百度还是要收敛一点。
- 难绷，还是很非常有民族自豪感的模型。有一些迷之本土化的成分。

## [秘塔AI搜索](http://metaso.cn/)

- 反应比较迅速
- UI 相对比较简洁
- 交互逻辑不是很直观，深入和追问等功能令人混淆
- 对搜索结果的选取逻辑还有待优化，比如我问刚刚发布的小米 15，依然基于前几个月的爆料贴进行分析。
- 尝试搜索 **thank you for inventing js** 这个 MEME 无果
- 这是出自什么动画片的台词：只要你能回来，我什么都愿意做。这个问题他倒会，看来只是不会外国梗？
- 搜索 JS Signal 引援的资料以国内为主
- 你能告诉我怎么提取长沙市的住房公积金吗。总体回答是对的，但是步骤指引并不直观。
- 印象 4/5

## [ying.baichuan-ai.com](https://ying.baichuan-ai.com/)

- UI 是类似聊天机器人那种
- 我感觉响应没有上面那家快，应该说比较慢
- 问了一下 3000 元手机推荐，感觉谁信谁上当
- 我问他 15 Pro 他同样被我误导了
- 对 JS Signal 的资料也是国内为主
- 印象 2/5

## [文心一言](https://yiyan.baidu.com/)

- UI 好脏乱差啊这个，符合百度的品味
- 松鼠桂鱼的问题修了。不过他死活不肯提供实拍照片或者访问链接。
- 翻译功能需要切选项卡，而且还只有几种主流语言。
- 印象 1/5

## [Perplexity](https://www.perplexity.ai/)

- 我一直在用的。上面几个国内竞品应该是学的这个，
- 他的 UI 这个颜色还是蛮好的，现在左侧经常有自己的广告不过无伤大雅
- 代理访问有时候会弹 Cloudflare，而且不定期会断线，需要手动刷新
- 这个是真正的互联网搜索，凭这点就跟上面几个不是同一位面了。
- 当然互联网要是搜不到的东西也没辙。
- 印象 5/5

## [chat.mistral.ai](https://chat.mistral.ai/chat)

- 总部在巴黎的一家 AI 企业
- 说实话我没感觉到他跟 OpenAI 有代际差别
- 问了他一个猜猜我来自哪里的问题，能回答出答案。
- 似乎是不能联网的，反正我问他 LESS 和 SCSS 哪个更流行，应该是给了语料库的答案。
- 印象 4/5

## [海螺AI—MiniMax 旗下生产力产品，你的AI伙伴，10倍速提升工作学习效率](https://hailuoai.com/)

- 也是比较简约的 UI
- 在猜猜我来自哪里的这个问题上，体验极差
- 对梗的解释也完全扑街
- 问了下怎么评价小米手机 15，发现他都没联网查梗。
- 顺便问了下手机推荐，都是几年前的。呃呃
- 印象 2/5

## [智谱清言](https://chatglm.cn/)

- 我依稀记得我用这个东西的原因是他接住了所有我问的网络梗
- 跟竞品相比 UI 就不是很简洁
- 印象 3/5

## [‎Gemini - chat to supercharge your ideas](https://gemini.google.com/)

- 起个大早赶个晚集的 Google 大模型
- 响应速度还是挺快的，是真的快
- 碰到了 bug，好像他不能存储任何人格相关的信息，也不能假设自己的人格。
- 印象 3/5

## [DeepSeek](https://chat.deepseek.com/)

- 明确不能联网，数据截止去年 10 月。
- 评分很高，暂时不知道为什么。
- 不过他们的 token 确实很便宜。
- 感觉有点小聪明，但是大事有点糊涂。。
- 印象 3/5

## [天工AI - 搜索更深度，阅读更多彩](https://www.tiangong.cn/)

- 这家好像是专做文档分析和联网检索的
- 基本问题跟上面几家是一样的，速度还不是很快
- 印象 2/5

## [coral.cohere.com](https://coral.cohere.com/)

- UI 的框框有点多
- 这个我感觉真跟前面几个有代差了
