# 如果能重来，我要这样学

才怪。

~~学习，学个屁。我直接梭哈比特币，英伟达，CPO~~

翻出来我的一些压箱底的教程，按学习先后排序。

## 入门篇

### 智慧上网

堪称中国人的赛博性教育，最有价值却也是最讳莫如深的一课。

很惭愧，这方面我也没找到特别好的教程。

### 代码

[Interneting Is Hard](https://internetingishard.netlify.app/) 可能是世界上最好的 HTML/CSS 入门教程没有之一。另外HTML/CSS 可能又是世界上最适合新手学代码的语言。这俩甚至都不是编程语言，恰好可以用来熟悉编辑器和 shell 等工具的使用。

[The Modern JavaScript Tutorial](https://javascript.info/) 摩登爪哇脚本教程。这个教程对 JS 的各类历史遗留问题都一笔带过了，学完上面的教程后，再来手搓个 todolist 应该不在话下。我觉得比较遗憾的地方就是这个教程对异步这一章的讲解还是有些隔靴搔痒。

### 工具链（版本控制）

值得在这个阶段谈得上系统学习的工具链大约也就 Git 吧。

[简介 - Git教程 - 廖雪峰的官方网站](https://liaoxuefeng.com/books/git/introduction/index.html) 廖老师的 Python 教程最有名，但是 Python 教程反而其实是写的最差的。他的 Git 教程（顺便说一下还有 Java）反倒写得还蛮不错。Git 是为 Linux Kernel 这样全球各地开发者参与构建的大型项目而生，它的复杂性是有必要的。这份教程比较好地帮助初学者屏蔽了那些复杂性和噪音，得以快速上手这个最流行的版本控制系统。

[Pro Git](https://git-scm.com/book/en/v2) 如果想了解进阶操作可以翻看这本书。

## 计算机科学基础篇

[CS自学指南](https://csdiy.wiki/) 应该是非常有名了。能在大学前两三年把这些林林总总的教程全学完那真是**超天才**。实际上能学透吃透下面这一部分，就可能超过十成甚至九成的应届生了。有如此扎实的基本功，进可备战春招，~~退则备战研究生招考。~~

* 微积分
* 线性代数
* 概率论
* 离散数学
* 数据结构与算法
* 计算机组成原理
* 操作系统
* 计算机网络
* 数据库原理
* ...（任何 1 - 2 门你感兴趣的科目）

[计算机专业学习路线 | HackWay技术学习路线](https://hackway.org/) 可以跟上面那个对照，筛选出二者交叉的课程看一看。

## 就业

[Developer Roadmaps - roadmap.sh](https://roadmap.sh/) 然后选择你的职业路线吧，少年

### Web 前端

[TypeScript 入门教程](https://ts.xcatliu.com/) 这本小册子在官方文档上适当做了减法，尽量做到每个章节不穿插别的章节的内容。

[JavaScript 教程](https://wangdoc.com/javascript/) 和
[ES6 入门教程](https://es6.ruanyifeng.com/) 两份教程均出自阮一峰老师。坦白说，行文比较干。在今天反而很少能找到把 JS 语言拆成了 ES5 和 ES6+ 两个版本分别讲解的教程了，也算是提供了一种独特的视野吧。~~年纪轻轻学什么 ES5 啊。~~

[Guides Archive - CSS-Tricks](https://css-tricks.com/guides/#top-of-site) 饱受好评的 CSS 系列教程

[Exploring JavaScript](https://exploringjs.com/) 挺好的一本书，个人觉得比红宝书要详实。作者说是 AI 天天来爬他的网站，不堪其扰把免费版下线了。目前只能去电商购买。

[huzidaha/react-naive-book: 开源、免费、专业、简单的 React.js 在线教程](https://github.com/huzidaha/react-naive-book) 虽然过时了，但是我再也没能找到一份教程，能像它一样循序渐进地向新手解释明白 React.js 等响应式组件库的理念。

### 服务端

这方面我是纯外行，献丑了。

[Beej's Guide to C Programming](https://beej.us/guide/bgc/html/index-wide.html) 一本很厚的字典级教程。我只读过目录，看上去是面面俱到的。

[Rust 语言之旅 - Let's go on an adventure!](https://tourofrust.com/TOC_zh-cn.html) 这个也是一直在收藏夹吃灰的教程。这个教程的特色是每个小节有交互式练习题。

[SQLBolt - Learn SQL - Introduction to SQL](https://sqlbolt.com/) 依旧是有交互式练习题的一本小册子。

[Go语言101](https://gfw.go101.org/article/101.html) 这个阅读体验像是通俗版的语言规范，但是部分章节依然晦涩。

[Go 语言原本](https://golang.design/under-the-hood/) /
[Learn Go with tests | learn-go-with-tests](https://studygolang.gitbook.io/learn-go-with-tests) /
[Go by Example](https://gobyexample.com/) 这三本坦白说只读过目录。

### 笔试

[Grind 75 - A better Blind 75 you can customize, by the author of Blind 75](https://www.techinterviewhandbook.org/grind75/) 刷完这个初级和中级难度就应该差不多了吧。
